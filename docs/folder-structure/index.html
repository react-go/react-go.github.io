目录结构

@react-go/scripts 对项目的目录结构有一定的要求：

my-app
|-- node_modules
|-- dist
|-- public
|-- index.html
|-- src
|-- index.js
|-- App.jsx
|-- package.json
|-- README.md

以下文件必须存在：

  • public/index.html - 项目的页面模板文件
  • src/index.js - 项目入口文件
